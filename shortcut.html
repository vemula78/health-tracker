Health Sync shortcut

A one-tap iOS Shortcut that copies today's Apple Health data (including everything your Watch recorded) to the clipboard as JSON, ready for the Health sync screen in Health Log. Build it once — about 5 minutes. Tap any key name on this page to copy it.

1

Create the shortcut

Open the Shortcuts app → tap + → rename it Health Sync (tap the title at the top).

2

Today's steps

Add the action Find Health Samples (search "health samples"). Configure it:

  • Tap Add Filter → Type is Steps
  • Tap Add Filter again → Start Date is today

Then add Calculate Statistics directly below: Operation Sum, Input Health Samples.

3

Distance, energy, exercise

Repeat step 2 three more times (each: Find Health Samples with the two filters, then Calculate Statistics → Sum) for these types:

  • Walking + Running Distance
  • Active Energy
  • Exercise Minutes

Tip: long-press an action → Duplicate to copy the pair, then just change the Type filter.

4

Resting heart rate

Once more: Find Health Samples with Type Resting Heart Rate + Start Date is today, then Calculate Statistics with Operation Average.

5

Workouts (optional)

Add Find Workouts with filter Start Date is today. Then:

  • Add Repeat with Each (input: Workouts)
  • Inside the repeat, add a Text action containing three magic variables from Repeat Item, separated by pipes:
    Activity Type|Duration|Total Active Energy
    (tap each blue variable token to insert it; set Duration's unit to minutes if asked)
  • After End Repeat, add Combine Text: combine Repeat Results with New Lines

Skip this step entirely if you don't want workouts — the importer doesn't mind.

6

Today's date

Add Format Date (it inserts Current Date automatically). Set Date Format to Custom and type exactly: yyyy-MM-dd

7

Build the dictionary

Add a Dictionary action and create one Text item per row below. For each value, tap the field → Select Magic Variable and pick the matching result from the actions above.

Key (tap to copy)Value (magic variable)
dateFormatted Date (step 6)
stepsStatistics from step 2
distanceKmStatistics — distance sum
activeEnergyKcalStatistics — active energy sum
exerciseMinStatistics — exercise minutes sum
restingHRStatistics — resting HR average
workoutsTextCombined Text (step 5, if built)

Skip any row you're not tracking — missing keys are simply ignored.

8

Copy as JSON

Add Get Text from Input (input: the Dictionary — this turns it into JSON text), then Copy to Clipboard. Done — tap the shortcut to test it.

9

First run & daily use

The first run asks permission to read each Health data type — allow each once. The clipboard should then hold something like:

{"date":"2026-07-05","steps":"8,234 count",
"distanceKm":"6.2 km","activeEnergyKcal":"512 kcal",
"exerciseMin":"34 min","restingHR":"58 count/min",
"workoutsText":"Running|32 min|310 kcal"}

Units and commas in the values are fine — the importer strips them. Then open Health Log → Health sync → Paste from clipboard & import. Re-syncing the same day updates it — no duplicates.

Make it effortless: add the shortcut to your home screen next to Health Log (shortcut settings → Add to Home Screen), or set up an automation (Shortcuts → Automation → Time of Day, e.g. 9 pm, "Run Immediately") so the day's data is already on the clipboard every evening.

Exact action and filter names can vary slightly between iOS versions. If a sum comes back empty, check that the Start Date filter says "is today" and that the data type has permission in Settings → Privacy & Security → Health.
